## Installation

1. Repository klonen:
  ```bash
  git clone <repository-url>
  ```
2. Abhängigkeiten installieren:
  ```bash
  npm install
  ```

## Nutzung

Starten Sie die Anwendung mit:
```bash
npm start
```
